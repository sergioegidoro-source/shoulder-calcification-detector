Input Image

Input Processing & Multi-Scale Pipeline
+
  • Accepted formats: DICOM, JPG and PNG radiographic images.
  • U-Net ROI extraction → 300×300 px (CNN classification): The humerus is segmented to center the image on the humeral head and the region of interest is cropped and resized to 300×300 px.
  • YOLO detection input → 896×896 px: The same ROI is resized to 896×896 px and used for calcification detection and localization with bounding boxes.
  • Standardization: All inputs are converted to RGB and undergo intensity normalization to ensure stable and comparable results.

Visual Interpretability (Grad-CAM)
+

This heatmap visualizes the model’s attention, highlighting the regions of the X-ray that most influenced the prediction.

  • Warmer colors: indicate areas with stronger influence on the prediction.
  • Cooler colors: represent regions with lower contribution.
  • Helps clinicians understand where the model focused when estimating the likelihood of calcific tendinopathy.
  • Interpretation aid: supports clinical review but is not a definitive lesion outline.

U-Net Segmentation

Anatomical Segmentation (U-Net)
+

This mask is generated by a U-Net segmentation network that isolates the humerus / humeral head from the surrounding anatomy.

  • Binary segmentation mask: each pixel is classified as bone (1) or background (0).
  • Helps remove irrelevant structures and reduce background noise.
  • Ensures the analysis focuses on the clinically relevant anatomical region.
  • Enables subsequent preprocessing steps such as automatic ROI centering and cropping.

Auto-Crop ROI

Mask-Guided AutoCrop (ROI)
+

The Region of Interest (ROI) is automatically computed from the segmentation mask to center the analysis on the joint space.

  • The humeral head location is used to determine the optimal crop region.
  • Produces a standardized input focused on the subacromial space.
  • Reduces irrelevant background and improves diagnostic sensitivity.
  • Ensures consistent input geometry for the downstream AI models.

YOLOv8 Precision

Object Detection (YOLOv8)
+

This module performs object detection to localize calcific deposits in positive cases.

  • Draws a bounding box around the suspected calcification.
  • Provides a confidence score indicating detection reliability.
  • Estimates the most likely affected tendon: supraspinatus, infraspinatus, or subscapularis.
  • Helps clinicians quickly identify where the calcification is located.

Decision-support tool only — final interpretation remains clinical.

Calcification Type Analysis

Hybrid Analysis (U-Net + Radiomics + ML)
+

The calcification region is segmented using U-Net (300px resolution). Radiomic descriptors are extracted from the segmented region, and a Logistic Regression classifier determines whether the morphology corresponds to Type I-II or Type III.
